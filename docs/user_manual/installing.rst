.. currentmodule:: sparse

Installing
==========
:obj:`sparse` can be obtained from pip via

.. code-block:: bash

   pip install sparse

You can also get :obj:`sparse` from its current source on GitHub, to get all
the latest and greatest features. :obj:`sparse` is under active development,
and many new features are being added. However, note that the API is currently
unstable at this time.

.. code-block:: bash

   git clone https://github.com/mrocklin/sparse.git
   cd ./sparse/
   pip install .
